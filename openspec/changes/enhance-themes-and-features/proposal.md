# 增强 weixin-md：多主题配色 + 外链引用 + 图表公式支持

## 问题

当前 weixin-md 只有一套写死的主题样式，无法满足不同风格的排版需求。同时缺少以下微信公众号写作中常见的功能：

- 外链在微信中会触发安全提醒弹窗，体验差
- 不支持 Mermaid 图表、数学公式、脚注、ruby 注音等进阶 markdown 语法
- 参考 baoyu-markdown-to-html 的 4 主题 + 13 配色 + 外链转底部引用方案，weixin-md 在功能完整度上有明显差距

## 目标

1. 支持多主题（4 套）+ 多配色（13 种预设），主题和配色可任意组合
2. 外链自动转为底部数字引用（`文字[1]` + 底部列表），微信内链保持原样
3. 支持 Mermaid 图表渲染预览 + 下载 PNG
4. 支持 KaTeX 数学公式渲染（行内 + 块级）+ 块级公式下载 PNG
5. 支持脚注语法（`[^1]`）
6. 支持 ruby text 注音语法（`{汉字}(hànzì)`）
7. 包含不可复制内容时，titlebar 显示警告图标提示用户

## 非目标

- 不做在线图床上传功能
- 不做微信公众号 API 对接发布
- 不做编辑器侧的 Mermaid/KaTeX 语法高亮（只做预览渲染）
- 不改变现有的 marked + highlight.js 技术栈基础
